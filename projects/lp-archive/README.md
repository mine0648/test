# LP Archive - ランディングページデザインギャラリー

rdlp.jp と codef.jp を参考にして作成したLPアーカイブサイト

## 機能

- カラーフィルター（8色）
- イメージタグフィルター（16種類）
- カテゴリーフィルター（12カテゴリー）
- レスポンシブデザイン
- スムーススクロール
- インタラクティブなフィルタリング

## 参考サイト

- https://rdlp.jp/lp-archive/
- https://codef.jp/

## ファイル構成

- `index.html` - メインHTML
- `style.css` - スタイルシート
- `script.js` - JavaScriptロジック
